# Inf'Eau

Projet d'analyse de la qualité de l'eau réalisé dans le cadre du Challenge 48h de Rennes Ynov Campus.

## Project setup

```
# yarn
yarn

# npm
npm install
```

### Compiles and hot-reloads for development

```
# yarn
yarn dev

# npm
npm run dev
```

### Compiles and minifies for production

```
# yarn
yarn build

# npm
npm run build
```

### Lints and fixes files

```
# yarn
yarn lint

# npm
npm run lint
```
